**HW2_OOP_Testing_Git**

**Описание проекта**

Проект выполнен в рамках домашнего задания №2 по дисциплине «Технологии программирования».

Разработано система для управления рецептами. Это консольное приложение, которое позволяет создавать блюда, добавлять их в "рецепты", масштабировать порции и генерировать список покупок.

* создавать ингредиенты;
* создавать рецепты и управлять их составом;
* масштабировать количество ингредиентов в рецептах;
* формировать список покупок на основе нескольких рецептов;
* объединять списки покупок;
* работать с диетическими рецептами.

Также реализованы автоматические тесты с использованием библиотеки pytest.

git clone https://github.com/Yar855/HW2_OOP_Testing_Git.git
